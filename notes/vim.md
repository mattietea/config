# Vim

## Keyboard Shortcuts

Moving lines

```
test(*)
t*
test*
```

Words

`w` - move to start or end of word

```
(*)word* *again*
```

`b` - move backwards to start or end of word

```
*word* word*.* word*.(*)
```

`e` - move to end of words

```
(*)word* word*.* word*.*
```
